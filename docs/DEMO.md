## 0b. Demo-mode env flags (must be set on workers)

These bypass three known 0G testnet / Uniswap deployment gaps. Each is logged when triggered, so the caveat is auditable in the recorded terminal output.

| Flag | Reason |
|------|--------|
| `STORAGE_FALLBACK_ON_FAIL=1` | 0G Storage flow contract `submit` reverts with `require(false)` on Galileo testnet (root cause: 0G testnet flake). Signal writes envelope to `/tmp/argus-storage/<root>.json`; execution reads from same path on download failure. Same-host guarantee. |
| `ZEROG_DEV_BYPASS_VERIFY=1` | 0G Compute provider's chatID signature endpoint returns `getting signature error` for a window after inference. Bypass keeps `isVerified=true` and logs `verify_bypassed` warning so it's visible in the demo. |
| `QUOTER_OPTIONAL=1` | Uniswap V3-style QuoterV2 not deployed at canonical address on Unichain Sepolia. With this flag, execution skips quote and uses `amountOutMin=1` (testnet demo). |
| `UNICHAIN_SEPOLIA_RPC=$UNICHAIN_SEPOLIA_RPC_BACKUP` | Public `sepolia.unichain.org` drops txs in mempool. Use Alchemy RPC for both submission and read paths. |

### 2b. AXL daemons (local)

```bash
bash scripts/start-nodes.sh
sleep 5
curl -fsSm 3 http://127.0.0.1:9002/topology | jq -r .our_public_key
```

Expect both pubkeys printed (note them тАФ pass to workers + shim) and at least one peer entry with `up: true`. The Yggdrasil pubkeys regenerate per local launch unless persistent keys are wired into `start-nodes.sh`. If pubkeys differ from previous run, update worker env (see 2d).

Logs at `/tmp/argus-axl-a.log`, `/tmp/argus-axl-b.log`. Stop with `bash scripts/start-nodes.sh stop`.

### 2c. Shim (local)

```bash
NODE_A_PUBKEY=$(curl -sS http://127.0.0.1:9002/topology | jq -r .our_public_key)
nohup bash -c 'set -a; . ./.env; set +a; \
  export SIGNAL_PEER='"$NODE_A_PUBKEY"' AXL_NODE_A_API=127.0.0.1:9002 SHIM_PORT=8790; \
  npm --prefix shim start' > /tmp/shim.log 2>&1 &
sleep 4
curl -fsSm 3 http://127.0.0.1:8790/health
```

Expect `{"status":"ok","pending":0}`.

Port `8787` is taken by Docker on this machine; we use `8790`. If that's also taken, pick any free port and update `SHIM_PORT` + `SHIM_URL` (2d).

### 2d. Workers (local)

```bash
NODE_A_PUBKEY=$(curl -sS http://127.0.0.1:9002/topology | jq -r .our_public_key)
NODE_B_PUBKEY=$(curl -sS http://127.0.0.1:9003/topology | jq -r .our_public_key)

# Terminal C тАФ signal worker (sends to Node B; talks to AXL Node A)
nohup bash -c 'set -a; . ./.env; set +a; \
  export UNICHAIN_SEPOLIA_RPC="$UNICHAIN_SEPOLIA_RPC_BACKUP" \
         SHIM_URL=http://127.0.0.1:8790 \
         EXECUTION_PEER='"$NODE_B_PUBKEY"' \
         STORAGE_FALLBACK_ON_FAIL=1 ZEROG_DEV_BYPASS_VERIFY=1 QUOTER_OPTIONAL=1; \
  npm --prefix signal start' > /tmp/signal.log 2>&1 &

# Terminal D тАФ execution worker (talks to AXL Node B; verifies messages from Node A)
nohup bash -c 'set -a; . ./.env; set +a; \
  export UNICHAIN_SEPOLIA_RPC="$UNICHAIN_SEPOLIA_RPC_BACKUP" \
         EXECUTION_PEER='"$NODE_A_PUBKEY"' \
         STORAGE_FALLBACK_ON_FAIL=1 ZEROG_DEV_BYPASS_VERIFY=1 QUOTER_OPTIONAL=1; \
  npm --prefix execution start' > /tmp/execution.log 2>&1 &

# Wait until both started
until grep -qE "signal.started" /tmp/signal.log && grep -qE "execution.started" /tmp/execution.log; do sleep 1; done
echo OK
```

For on-camera recording, replace `nohup ... > /tmp/*.log 2>&1 &` with foreground `npm --prefix signal start` / `npm --prefix execution start` in two real terminals so log lines stream visibly.

### 2e. Dry-run (do this once, 10 min before recording)

